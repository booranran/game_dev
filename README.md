# 팔꿈치 사회 (Elbow Society)

> 평소 지하철을 탄 경험이 많은 플레이어가, 붐비는 지하철 안에서 자리 양보와 무시 사이의 선택을 반복하며 체력과 배려 사이의 딜레마를 경험하는 소셜 생존 시뮬레이션 게임입니다.

- **장르**: 시뮬레이션
- **플랫폼**: Mobile (Android)
- **엔진**: Unity 2D

## 게임 소개

혼잡한 대중교통 문화를 게임적으로 재해석한 소셜 생존 시뮬레이션입니다. 서로 다른 특징을 지닌 캐릭터를 선택해 전철 안에서 체력, 배려, 컨디션을 관리하며 목적지까지 도착하는 것이 목표입니다.

지하철 안에서 누구나 한 번쯤 겪어봤을 현실적인 상황들 — 노약자/임산부에게 자리를 양보할지 말지, 빈자리를 두고 다른 승객과 눈치싸움을 벌일지, 서서 버틸지 등 — 을 게임 시스템으로 풀어내, 윤리적 딜레마를 가볍고 유쾌하게 체험할 수 있도록 만들었습니다.

## 플레이 방식

### 기본 흐름

```
역 정차 → 이벤트 발생 → 양보/무시 선택 (또는 미니게임) → 체력·배려점수 변화 → 다음 역으로 이동 → 반복
```

매 정거장마다 다양한 이벤트가 발생합니다.

- **양보 대상 NPC 등장**: 노약자나 임산부에게 자리를 양보할지, 무시할지 선택합니다. 양보하면 배려점수를 얻고 일어서야 하며, 무시하면 자리는 지키지만 죄책감(페널티)이 남습니다.
- **빈자리 발생**: 자리가 비면 눈치게임을 통해 그 자리를 차지할 수 있습니다. 다른 승객과 타이밍 경쟁이 붙을 수도 있습니다.
- **미니게임 발생**: 자리를 지키거나 서있는 동안 상황에 따라 미니게임이 발생합니다.

### 캐릭터

| 캐릭터 | 특징 |
|---|---|
| 대학생 | 체력적으로 여유 있지만, 배려점수를 모으기 어려움 |
| 직장인 | 체력이 부족하지만, 배려점수가 잘 쌓임 |

### 호선 선택 (난이도)

| 호선 | 특징 |
|---|---|
| 9호선 | 출퇴근 시간대, 혼잡해서 한 번 양보하면 다시 앉기 어려움 → **체력 관리**가 핵심 |
| 7호선 | 애매한 시간대, 자리가 있는 듯 없는 듯 — 눈치게임 요소가 강조됨 → **배려점수 관리**가 핵심 |

### 미니게임

| 미니게임 | 발동 조건 | 내용 |
|---|---|---|
| 눈치게임 | 빈자리 발생 시 | 빈자리를 관찰하고, 다른 승객과 타이밍 경쟁을 벌여 자리를 선점합니다 |
| 팔꿈치게임 | 앉아있을 때 | 밀려오는 옆자리 승객을 게이지 타이밍에 맞춰 막아내고 자리를 지킵니다 |
| 가방 방어 게임 | 서있을 때 (사람이 많을 때) | 밀려드는 가방을 제때 막아내 컨디션과 체력을 지킵니다 |

### 핵심 자원

- **체력**: 서있으면 매 턴 소모되고(컨디션이 나쁠수록 더 많이), 앉아있으면 회복됩니다.
- **배려점수**: 양보를 선택하면 획득합니다. 서있던 턴 수(콤보)가 많을수록 더 많이 받을 수 있습니다.
- **컨디션**: 미니게임 결과나 NPC와의 상호작용에 따라 변하며, 체력 소모량에 직접 영향을 줍니다.

### 결말

체력이 0이 되면 **게임 오버**입니다. 목적지까지 무사히 도착하면 체력과 배려점수 조합에 따라 엔딩이 결정됩니다.

| 엔딩 | 조건 |
|---|---|
| 굿엔딩 | 체력 60% 이상 & 배려점수 60 이상 |
| 노멀엔딩 A | 체력 60% 이상 & 배려점수 60 미만 |
| 노멀엔딩 B | 체력 60% 미만 & 배려점수 60 이상 |
| 배드엔딩 | 체력 60% 미만 & 배려점수 60 미만 |
